Variables

Una variable es un lugar en la memoria de nuestra computadora para almacenar datos

Tipos de datos en JavaScript

  1. Primitivos: Se accede directamente al valor.
    • string
    • number
    • boolean
    • null
    • undefines
    • NaN
  2. Compuestos o Complejos: Se accede a la referencia del valor.
    • object = {}
    • array = [ ]
    • function () {}
    • Class {}
    • etc.

Fundamentos de JavaScript

var vs let

Las variables tienen un ámbito de existencia/bloque, lo que se conoce como el scope de las variables.

En las versiones anteriores a ECMAScript 6, solo se utilizaba var y el alcance de var es global por lo tanto al utilizar variables dentro de funciones con un bloque if, si se declaraba var dentro de un if esta variable tenia alcance global dentro de esa función.

Para solucionar esto, en la versión de ECMAScript 6 se agregó otro tipo de variable llamado "let" el cual su scope es el bloque de código donde se encuentre declarada.

Un bloque de JavaScript esta separado por llaves {}

En el navegador el objeto global es window mientras en node es global, con var uno accede a las variables con window.nombre y en node con global.nombre
