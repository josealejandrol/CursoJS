Cadenas de texto (strings)

Un string es una secuencia de caracteres.

Es un objeto que representa una serie de caracteres dentro de una cadena, según MDN.


Los objetos en JavaScript tienen dos atributos muy importantes: propiedades y metodos

  • Las propiedades van a ser atributos de un objeto, que nos dan información del objeto
    • Por ejemplo: La propiedad de un string es: String.length
  • Los metodos son acciones que el objeto hace.

La diferencia entre las propiedades y los metodos es:

  • Un metodo termina con parentesis (), dentro de esos parentesis puede que reciba ciertos datos para poder funcionar.
