Expresiones Regulares

Una expresión regular es una secuencia de caracteres que forman un patrón de busqueda y que principalmente es utilizado en la busqueda de cadenas de texto.

  • https://es.wikipedia.org/wiki/Expresi%C3%B3n_regular
  • https://developer.mozilla.org/es/docs/Web/JavaScript/Guide/Regular_Expressions

Las expresiones regulares no son algo exclusivo de JavaScript, cualquier tecnología que se de un lenguaje de programación, tiene que tener una implemetación de expresiones regulares.

Las expresiones regulares facilitan mucho la vida, por ejemplo:

  • Cuando quieras validar que un correo electrónico se haya escrito y tenga los parametros validos
  • Para validad un DNI
  • Para validad el RFC
  • Validar los 16 digitos de una tarjeta

Existen 2 métodos para las expresiones regulares:

  • .test() que devuelve un true o false dependiendo de la validación
  • .exec() que devuelve un array con información
