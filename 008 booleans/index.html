Booleanos Booleans

Este tipo de dato nos permite declarar una variable con 1 valor u otro unicamente, verdadero o falso: true o false.


Hay valores que aunque no sean declarados como booleanos con el valor true, tienden a ser verdaderos por ejemplo:

  • if(true)//true
  • if({})//true
  • if([ ])//true
  • if(42)//true
  • if("foo")//true
  • if(new Date())//true
  • if(-42)//true
  • if(3.14)//true
  • if(-3.14)//true
  • if(Infinity)//true
  • if(-Infinity)//true
