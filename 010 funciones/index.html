Funciones function

Las cosas por lo general en programación suceden dentro de funciones.

Una función es un bloque de código autocontenido, esto quiere decir que, tiene su propio contenido independientemente al scope o el ambito global.

Una función se define una sola vez y se puede ejecutar en cualquier momento.

Es un fragmento de código que se declara una vez y se puede reutilizar tantas veces como sea necesario.

Una función puede o no recibir parametros, que son datos que recibe separados por coma (,).

Puede devolver o no valores.


Las funciones particularmente en javascript son un objeto, son un tipo de dato muy especial.

Las funciones son tan poderosas, pueden recibir funciones y también pueden retornar la activación de una función

Se dice que las funciones son ciudadanos de primera clase porque pueden asignarse a un valor, y pueden pasarse como argumentos y usarse como un valor de retorno
