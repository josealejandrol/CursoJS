Módulos en Javascript

Los módulos en JavaScript significa que puedes mandar a llamar los archivos .js no solo desde un archivo, sino tambien desde un archivo JavaScript mandar a llamar otro.

Los archivos js se pueden importar dentro de las etiquetas head de nuestro archivo html si necesitamos cargar el código antes de que el navegador lea el contenido del body

También pueden importarse dentro de la etiqueta body, antes del cierre de la etiqueta body porque el código js es bloqueante y si tus archivos pesan mucho puede hacer que tarde en cargar

Para que los archivos js tengan la capacidad de llamar otros archivo js hay que declarar en la etiqueta script donde se invoca el archivo js, agregar el atributo type="module"
