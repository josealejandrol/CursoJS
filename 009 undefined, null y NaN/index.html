undefined, null y NaN

null, undefined y NaN son casos muy particulares de JavaScript

Tanto como null y undefined van a representar un valor ausente, es decir una variable que no tiene valor, pero la gran diferencia es que mientras que:

  • undefined no ha sido inicializada.
  • null es un valor especial que indica que la variable esta vacía, es un valor intencionalmente asignado por el programador.
  • también undefined es un valor asignado por javascript, cuando detecta que una variable no tiene valor.

NaN, Not a Number. Es arrojado por JavaScript cuando al hacer una operación aritmetica uno de los 2 datos que se vayan a sumar por ejemplo no sea un número.
