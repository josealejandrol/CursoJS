JavaScript

ISOMORFISMO

Significa que puedes hacer una aplicación de principio a fin con una tecnología.


Hoy JavaScript, es el único lenguaje capaz de ejecutarse en las 3 capas de una aplicación:

  1. Frontend (JavaScript).
  2. Backend (Node.js).
  3. Persistencia de Datos (MongoDB, Couch DB, Firebase, etc).

Con JavaScript puedes:

  • Diseño y Desarrollo Web.
  • Hacer Videojuegos.
  • Experiencias 3D, Realidad Aumentada, Realidad Virtual.
  • Controlar Hardware (Drones, robots, electrodomésticos).
  • Aplicaciones Híbridas y Móviles.
  • Machine Learning.
  • etc.

JavaScript NO es Java

Características:

  • Lenguaje de Alto Nivel.

    • No nos tenemos que estar preocupando por gestionar la memoria.

  • Interpretado.
  • Dinámico.

    • No necesita una fase de compilación y luego pasar a una fase de ejecución.
    JavaScript se compila en tiempo de ejecución

  • Débilmente Tipado.

    • Cuando creas una variable no es necesario declararla con un tipo de dato.

  • Multi paradigma.

    • Podemos usar Programación Estructurada, POO, Programación Funcional

  • Sensible a MAYÚSCULAS y minúsculas.

    • Si yo declaro una variable PERRO o perro, son 2 variables diferentes.

  • No necesitas los puntos y comas al final de cada línea.

Escritura de código:

  • Los identificador deben comenzar con:
    • Una letra o
    • Un signo de dolar $ o
    • Un guión bajo _
    • Nunca con números o caracteres especiales
  • Usa snake_case en:
    • Archivos:

      • mi_archivo_javascript.js

  • Usa UPPER_CASE en:
    • Constantes:

      • const UNA_CONSTANTE = 'Soy una constante'

      PI = 3.141592653589793

  • Usa UpperCamelCase en:
    • Clases:
  • Usa lowerCamelCase en:
    • Objetos:

      • const unObjeto = {}

    • Primitivos:

      • let unaCadena = 'Hola Mundo'

    • Funciones:

      • function holaMundo(){}

    • Instancias:

      • const ajax = new XMLHttpRequest()

Ordenamiento de código:

  1. IMPORTACIÓN DE MÓDULOS.
  2. DECLARACIÓN DE VARIABLES.
  3. DECLARACIÓN DE FUNCIONES.
  4. EJECUCIÓN DE CÓDIGO.

Tipos de datos en JavaScript:

  1. Primitivos: Se accede directamente al valor.
    • string
    • number
    • boolean
    • null
    • undefines
    • NaN
  2. Compuestos o Complejos: Se accede a la referencia del valor.
    • object = {}
    • array = [ ]
    • function () {}
    • Class {}
    • etc.
