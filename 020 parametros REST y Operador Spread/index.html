Parámetros REST & Operador Spread

Son nuevas caracreristica que nos trajo ES2018 o ES9

  • Parámetros REST

    • Son una forma de virtualmente de ir agregando parámetros infitinios que ya sea a una función o dentro de una variable. Imaginate que tienes un arreglo y no sabes cuantos valores vas a recibir, posteriormente con esos valores vas a hacer operaciones o ciertos procesos.

    Para definir los parámetros REST hay que definirlos poniendo 3 puntos suspensivos antes del nombre donde están guardados esos posibles valores infinitos:
    ...nombreVariable

  • Operador Spread

    • Nos va a permitir cuando una expresión la tengamos que expandir, tengamos que almacenar múltiples argumentos o elementos lo podamos hacer.

    Con el spread operator vamos a poder agregar nuevos elementos a un array o un objeto
