Métodos estáticos, getters y setters

Todas las clases generadas en Javascript son públicas, en Javascript no se pueden hacer privados los atributos y métodos de una clase.

Un método estático es aquel que se puede ejecutar sin la necesidad de instanciar la clase, es decir, sin la necesidad de crear un elemento que herede la clase padre,

En javascript no existe public ni private para una clase, todas son publicas y no necesita poner la palabra public antes de declarar la clase
